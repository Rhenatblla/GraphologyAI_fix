STEP 1: SETUP MONGODB (Pilih salah satu)
────────────────────────────────────────
  
  A. LOKAL (Recommended):
     • Download: https://www.mongodb.com/try/download/community
     • Install dengan default settings
     • Jalankan: net start MongoDB (Command Prompt as Admin)
     • .env: MONGO_URI=mongodb://localhost:27017/graphology-ai
  
  B. CLOUD (Atlas):
     • Sign up: https://www.mongodb.com/cloud/atlas
     • Create M0 Sandbox cluster (free)
     • Get connection string
     • .env: MONGO_URI=mongodb+srv://user:pass@cluster.mongodb.net/graphology-ai


STEP 2: CREATE ADMIN ACCOUNT
────────────────────────────
  
  cd Capstone/backend
  npm install
  node seeder.js -admin
  
  ✅ Output:
     Admin berhasil dibuat!
     Email: admin@graphology.com
     Password: admin123456


STEP 3: RUN BACKEND & FRONTEND
──────────────────────────────
  
  TERMINAL 1 - Backend:
    cd Capstone/backend
    npm start
    
    ✅ Expected: Server started on port 5000
  
  TERMINAL 2 - Frontend:
    cd Capstone/frontend
    npm run dev
    
    ✅ Expected: Local: http://localhost:3000

🐛 TROUBLESHOOTING CEPAT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "MongoDB connection failed"
   → Run: net start MongoDB (Windows)
   → Atau check MongoDB running: Get-Service MongoDB

❌ "User not found" saat login admin
   → Run: node seeder.js -admin
   → Check database di MongoDB Compass

❌ Login redirect ke /homeanalisis bukan /admin
   → Check backend response punya role: "admin"
   → Clear localStorage: localStorage.clear()
   → Login lagi

❌ "CORS error" atau "Failed to fetch"
   → Check backend running di port 5000
   → Check frontend di port 3000
   → Verify .env MONGO_URI correct

❌ localStorage empty setelah login
   → Clear browser cache
   → Check AuthContext saving data
   → Login lagi

→ Untuk troubleshooting lengkap: ADMIN_SETUP_GUIDE.md
